1)Еремин Александр

2)Программа позволяет:
-добавлять расходы с указанием суммы, категории и даты;

-просматривать все записи в виде таблицы;

-фильтровать расходы по категориям и датам;

-подсчитывать общую сумму расходов за выбранный период;

-сохранять данные в формате JSON для последующего использования;

-загружать ранее сохранённые данные при запуске программы.

Программа обеспечивает проверку корректности ввода данных:

сумма должна быть положительным числом;

дата должна быть в формате ГГГГ‑ММ‑ДД.

3)Запуск программы. Запустите файл expense_tracker.py (убедитесь, что у вас установлен Python и библиотеки tkinter, json, datetime).

Добавление расхода:

введите сумму в поле «Сумма»;

выберите категорию из выпадающего списка («Еда», «Транспорт», «Развлечения», «Жильё», «Другое»);

укажите дату в формате ГГГГ‑ММ‑ДД в поле «Дата (ГГГГ‑ММ‑ДД)»;

нажмите кнопку «Добавить расход» — запись появится в таблице ниже.

Фильтрация данных:

выберите нужную категорию в выпадающем списке «Фильтр по категории» («Все» для отображения всех записей);

нажмите кнопку «Применить фильтр» — таблица обновится с учётом выбранного фильтра.

Подсчёт суммы за период:

введите начальную дату периода в поле «Начало периода (ГГГГ‑ММ‑ДД)»;

введите конечную дату периода в поле «Конец периода (ГГГГ‑ММ‑ДД)»;

нажмите кнопку «Подсчитать сумму за период» — под кнопкой отобразится общая сумма расходов за указанный период.

Автоматическое сохранение. Все добавленные расходы автоматически сохраняются в файл expenses.json в папке с программой. При следующем запуске данные будут загружены автоматически.

4) Добавление первого расхода
Шаг 1. Запустите программу — откроется окно с полями ввода и кнопками.

Шаг 2. Заполните поля:

Сумма: 750;

Категория: выберите «Еда» из выпадающего списка;

Дата: 2024-04-25 (формат ГГГГ‑ММ‑ДД).

Шаг 3. Нажмите кнопку «Добавить расход».

Результат:

запись с суммой 750 руб., категорией «Еда» и датой 2024‑04‑25 появится в таблице ниже;

поля ввода очистятся — можно добавлять следующую запись;

данные автоматически сохранятся в файл expenses.json.
